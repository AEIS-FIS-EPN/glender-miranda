#### Dependencies Installation

To install the project dependencies, run:

1. Install pnpm package (if you don't already have it installed).
```bash
npm install -g pnpm
```

2. Install the dependencies with pnpm package
```bash
pnpm install
```


#### Run

To run the project, use:

```bash
pnpm run dev
```
